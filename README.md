# Lighting-OpenGL
Лабораторная работа №14 по компьютерной графике: Освещение

## Задания
Для выполнения [задания](https://edu.mmcs.sfedu.ru/mod/assign/view.php?id=29813) надо сделать следующее:
- Сцена. Произвольная сцена, состоящая из нескольких (минимум пяти) разных объектов
- Источники освещения:
	1. Точечный источник света
	2. Направленный источник света (Глобальный )
	3. Прожекторный источник света
- Модели освещения
	1. Phong
	2. Toon Shading
	3. Любая другая модель из презентации (Кроме Блинна и Ламберта)

## Стек
* **Язык программирования**: C++17
* **Графический API**: OpenGL 3.3+
* **Оконная система**: SFML 2.6.1
* **Расширения**: GLEW для загрузки OpenGL функций
* **Шейдеры**: GLSL 330 core

## Первая сборка проекта:
``` bash
cd build
cmake ..    # ← Генерация (один раз)
make        # ← Компиляция
```

## После изменения файлов в **/3d-objects**:
```bash
make        # ← Пересборка (cmake уже не нужен)
```
